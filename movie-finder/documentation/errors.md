# ERROR CODE REFERENCING

__*This document describes the errors that can be returned by the program*__

## Login errors :
  

**no_user :**
    
    The user does not exist

**wrong_password :**
    
    The password is incorrect

## Registering errors :

**user_already_exists :**
    
    The user already exists

**weak_password :**
    
    The password is too weak
